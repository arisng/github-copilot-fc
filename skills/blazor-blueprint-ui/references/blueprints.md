# Blueprints

Source: https://blazorblueprintui.com/llms/blueprints.txt

## What blueprints are for

Blueprints are copy-paste Razor compositions built from BlazorBlueprint components plus Tailwind classes. They are the fastest starting point when the task is "build a login page", "give me a dashboard shell", or "create a sidebar layout".

They are not installable packages; copy the code into the app and adapt it.

## Fast selection guide

- Auth pages -> `login` or `signup`
- App shells/navigation -> `sidebar` or `navigation`
- Analytics/admin surfaces -> `apps`, `dashboard`, or `data`
- Generic business flows -> `forms`
- Landing pages -> `marketing`
- Storefront/product flows -> `ecommerce`

## Recommended workflow

1. Finish library setup first.
2. Pick the closest category.
3. Copy the Razor example from the upstream blueprint file.
4. Replace placeholder data, routes, and actions.
5. Re-theme with your app's CSS variables/utilities instead of rewriting the structure from scratch.
