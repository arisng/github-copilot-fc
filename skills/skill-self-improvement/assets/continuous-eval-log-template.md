# Continuous Eval Log

Use this log whenever a human rejects an output that passed the current binary assertions.

| Date | Skill Version | Case ID | Structural Score | Rejection Reason | Candidate New Assertion | Keep Human-Reviewed | Status |
|---|---|---|---|---|---|---|---|
| YYYY-MM-DD | v0 | case-01 | 10/10 | Replace with the human rejection reason | Replace with a candidate binary check | yes or no | open |

## Status values

- `open`: not yet reviewed
- `accepted`: converted into a new assertion
- `rejected`: intentionally kept as human review
- `superseded`: replaced by a better assertion
